diferentes tipos de inserção de String

Exsitem 3 tipos de inserção de String, aspas simples '', aspas duplas "" e cráse ``, as duas primeiras fazem praticamente as mesmas funções, que é englobar conjunto de caracteres como sendo um único. já a cráse tem uma funcionalidade se chama template String, que é uma String que aceita expressões incorporadas e também pode-se utilizar quebra de linhas.
