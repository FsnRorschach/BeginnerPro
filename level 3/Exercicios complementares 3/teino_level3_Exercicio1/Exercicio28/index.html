Exercicio Complementar 28

Qual a diferença entre Math.round() e Math.trunc()? Faça um exemplo.

Math.round() => Retorna um número inteiro mais próximo

Exemplos:

Math.round10(55.55, -1); // 55.6

Math.round10(55.549, -1); // 55.5

Math.round10(55, 1); // 60

Math.round10(54.9, 1); // 50

Math.round10(-55.55, -1); // -55.5

Math.round10(-55.551, -1); // -55.6

Math.round10(-55, 1); // -50

Math.round10(-55.1, 1); // -60

Math.round10(1.005, -2); // 1.01

Math.trunc() => Retorna apenas a parte inteira de um numero, podendo ser negativa ou positiva

Math.trunc(13.37); // expected output: 13

Math.trunc(42.84); // expected output: 42

Math.trunc(0.123); // expected output: 0

Math.trunc(-0.123); // expected output: -0
